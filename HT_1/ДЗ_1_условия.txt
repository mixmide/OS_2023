1. Написать программу.
Со стандартного потока ввода задаётся вещественное число ε.
Для ввода использовать функцию scanf :
double x;
scanf("%lf",&x); // &x — это адрес переменной x, в которую нужно записать считанное число Далее в цикле вводится вещественное число x. Для введённого x вычисляется ( метод касательных (Ньютона)) и печатается значение √x (квадратный корень из x).
Цикл завершается при вводе признака конца файла, EOF со стандартного ввода можно получить нажатием Ctrl-D .
Для вычисления квадратного корня используется итеративная формула
x0 = 1, xi+1 = (xi + x/xi)/2
Вычисления продолжаются, пока |xi − xi+1 | >= ε. Ответом является число xi+1 .
 
Каждый ответ должен размещаться на отдельной строке.
Ответ напечатайте, используя функцию printf, в нескольких вариантах : с помощью форматных преобразований %f, %g, %e, %.10g, с ограничением на количество знаков и т. п. (см. описание функции printf).
Стандартные математические функции использовать нельзя.
 
2.
Написать функцию вывода на экран целого числа в двоичном виде. Незначащие нули тоже выводите. Отрицательные числа должны выводиться в их внутреннем представлении (например, -1 выводится как последовательность из единиц).
Написать функцию, которая осуществляет циклический сдвиг беззнакового целого x на n позиций вправо или влево. Направление сдвига задается параметром функции. Измененное число вернуть в качестве результата.
 
В функции main() вводить целые числа, сдвигать, печатать в двоичном виде исходное число и результат сдвига.
